Role - Deploy Blueprint  (ravello.blueprint_deploy)
======================================================

This role will content blueprint into one or multiples applications
By default, it can also:
- Deploy all applications
- Generate ephemeral tokens for all applications
- Collect all public IP addresses and put them in a yaml file organized per application
- Collect all public IP addresses and create an inventory file
- Collect all ephemeral tokens url and put them in a yaml file
